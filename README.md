## Getting started  

```
$ git clone https://github.com/zacscoding/spring-websocket-random-chat.git  

$ mvn package  

$ java -jar target/spring-websocket-random-chat/jar

OR

$ mvn spring-boot:run

```  

connect : http://localhost:8080/
